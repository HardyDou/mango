# @mango/link-panel

兼容包。新项目请使用 `@mango/link-page`。

`@mango/link-panel` 继续导出 `LinkPanel`、`MangoLinkPanel`，它们与 `@mango/link-page` 的 `LinkPage`、`MangoLinkPage` 指向同一个真实页面组件。

## 推荐用法

```ts
import { MangoLinkPage } from '@mango/link-page';
import '@mango/link-page/style.css';
```

## 兼容用法

```ts
import { MangoLinkPanel } from '@mango/link-panel';
import '@mango/link-panel/style.css';
```
