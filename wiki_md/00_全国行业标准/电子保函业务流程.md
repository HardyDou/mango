# 电子保函业务流程数据流图

> 整理日期：2025-03-21

---

## 一、核心概念澄清

### 1.1 入围关系

| 关系 | 说明 |
|------|------|
| **担保公司入围交易系统** | 担保公司（您的平台）成为交易系统的合规服务商 |
| **出函机构接入担保公司** | 银行/保险/担保公司接入担保公司平台提供出函服务 |

### 1.2 出函方式

| 方式 | 说明 | 是否需要入围交易系统 |
|------|------|-------------------|
| **银行直开** | 银行直接入围交易系统出具保函 | 需要 |
| **分离式** | 担保公司入围，出函机构通过担保公司出具保函 | 不需要 |

### 1.3 支付模式

| 模式 | 说明 | 资金流向 |
|------|------|----------|
| **见费模式** | 先支付保费，后出函 | 投标人 → 担保公司平台 |
| **后费模式** | 先出函，后支付 | 投标人 → 担保公司平台 |
| **授信模式** | 银行授信，无需支付 | 银行垫付，后续追偿 |

## 二、投保申请流程

### 2.1 完整流程

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           投保申请流程                                       │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   ┌─────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐  │
│   │  投标人  │───▶│  交易系统   │───▶│ 担保公司平台 │───▶│  出函机构   │  │
│   └─────────┘    └─────────────┘    └─────────────┘    └─────────────┘  │
│        │              │                  │                  │            │
│        ▼              ▼                  ▼                  ▼            │
│   1.发起投保    2.转发投保请求   3.审核/分配机构    4.审核出函        │
│   选择项目/标段  加密传输         计算保费           返回结果           │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 2.2 数据流说明

| 步骤 | 发起方 | 接收方 | 数据内容 |
|------|--------|--------|----------|
| 1 | 投标人 | 交易系统 | 项目信息、标段信息、投标人信息 |
| 2 | 交易系统 | 担保公司平台 | 投保申请数据（加密） |
| 3 | 担保公司平台 | 出函机构 | 投保信息、保费计算结果 |
| 4 | 出函机构 | 担保公司平台 | 审核结果 |

### 2.3 投保状态

| 状态 | 说明 |
|------|------|
| SUBMITTED | 已提交（待审核） |
| REVIEWING | 审核中 |
| APPROVED | 审核通过（待支付） |
| REJECTED | 审核拒绝 |
| CANCELLED | 已取消 |

## 六、异常流程

### 6.1 放弃申请

```
投标人 → 申请放弃 → 担保公司平台 → 通知出函机构 → 更新状态
```

### 6.2 注销保函

```
交易系统（开标后） → 注销通知 → 担保公司平台 → 通知出函机构 → 更新状态
```

### 6.3 解密保函

```
交易系统（开标时） → 解密密钥推送 → 担保公司平台 → 出函机构解密 → 生成明文保函
```

### 6.4 理赔

```
招标人/监管部门 → 理赔通知 → 担保公司平台 → 出函机构审核 → 赔付 → 更新状态
```
