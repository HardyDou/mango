# PRD - 担保公司管理后台

## 1. 产品概述

### 1.1 产品背景

担保公司管理后台是电子保函系统面向金融机构（担保公司、银行、保险公司）的业务管理后台，用于处理投保审核、保函签发、退保处理、风险管理等核心业务。

### 1.2 产品定位

面向担保公司的业务管理平台，实现：
- 投保业务的审核与处理
- 保函的签发与管理
- 退保业务的处理
- 风险控制与管理
- 财务对账与报表

### 1.3 目标用户

| 角色 | 说明 | 主要功能 |
|------|------|----------|
| 风控岗 | 风控审核人员 | 投保审核、风控规则、黑名单 |
| 出函岗 | 保函签发人员 | 保函开具、签章、发放 |
| 财务岗 | 财务人员 | 发票管理、对账管理、报表 |
| 管理岗 | 管理人员 | 机构配置、用户管理、查看 |

## 3. 功能详细说明

### 3.1 工作台

#### 3.1.1 功能描述

业务人员登录后的首页，展示关键业务指标和待办事项。

#### 3.1.2 页面内容

| 区块 | 内容 |
|------|------|
| 顶部导航 | Logo、搜索、消息、用户 |
| 快捷入口 | 快速处理待办任务 |
| 关键指标 | 今日投保数、待出函数、待审核数、待退保数 |
| 待办事项 | 待处理投保申请列表（可点击处理） |
| 业务趋势 | 近7天投保数量趋势图 |
| 通知公告 | 系统通知、业务提醒 |

#### 3.1.3 指标卡片

| 指标 | 说明 |
|------|------|
| 今日投保 | 今日新增投保申请数 |
| 待出函 | 等待签发出函的保单数 |
| 待审核 | 待风控审核的申请数 |
| 待退保 | 待处理的退保申请数 |

---

### 3.2 投保管理

#### 3.2.1 功能描述

管理投保申请，支持查询、审核、导出等操作。

#### 3.2.2 列表页面

| 字段 | 说明 |
|------|------|
| 申请编号 | 系统生成 |
| 项目名称 | 对应项目 |
| 投标人 | 企业名称 |
| 保证金金额 | 担保金额 |
| 申请时间 | 提交时间 |
| 状态 | 待审核/审核中/已通过/已拒绝 |
| 操作 | 审核、查看详情 |

#### 3.2.3 状态说明

| 状态 | 说明 |
|------|------|
| 待审核 | 等待风控审核 |
| 审核中 | 正在审核中 |
| 已通过 | 审核通过，等待出函 |
| 已拒绝 | 审核拒绝 |

#### 3.2.4 审核功能

**审核表单**：

| 字段 | 说明 |
|------|------|
| 审核结果 | 通过/拒绝 |
| 审核意见 | 拒绝原因说明 |
| 额度检查 | 是否超过额度 |
| 风控规则 | 规则匹配结果 |

**审核操作**：
- 通过 → 进入出函流程
- 拒绝 → 通知投标人，可重新申请

#### 3.2.5 高级搜索

| 筛选项 | 选项 |
|--------|------|
| 状态 | 全部、待审核、审核中、已通过、已拒绝 |
| 时间范围 | 申请时间筛选 |
| 项目 | 按项目名称搜索 |
| 投标人 | 按投标人名称搜索 |

---

### 3.3 保函管理

#### 3.3.1 功能描述

管理已开具的保函，支持签章、发放、查询等操作。

#### 3.3.2 列表页面

| 字段 | 说明 |
|------|------|
| 保函编号 | 系统生成唯一编号 |
| 项目名称 | 对应项目 |
| 投标人 | 企业名称 |
| 保证金金额 | 担保金额 |
| 保费 | 实收保费 |
| 开具时间 | 签发时间 |
| 到期时间 | 失效时间 |
| 状态 | 出函中/已开具/已发放/已过期 |
| 操作 | 查看、下载、发放、作废 |

#### 3.3.3 状态说明

| 状态 | 说明 |
|------|------|
| 出函中 | 正在生成保函文件 |
| 已开具 | 保函已签发完成 |
| 已发放 | 已推送给投标人/交易系统 |
| 已过期 | 保函已到期 |

#### 3.3.4 保函签章

| 步骤 | 说明 |
|------|------|
| 1. 生成文件 | 根据模板生成OFD/PDF |
| 2. 电子签章 | CA证书签章 |
| 3. 时间戳 | 添加可信时间戳 |
| 4. 推送 | 推送给投标人/交易系统 |

#### 3.3.5 保函详情

| 区块 | 内容 |
|------|------|
| 基本信息 | 保函编号、项目信息、金额 |
| 投标人信息 | 企业信息、联系人 |
| 投保信息 | 投保时间、审核记录 |
| 机构信息 | 出函机构、费率 |
| 保函文件 | 预览、下载 |
| 操作记录 | 状态变更时间轴 |

### 3.4 退保管理

#### 3.4.1 功能描述

处理退保申请，包括审核和执行退保。

#### 3.4.2 列表页面

| 字段 | 说明 |
|------|------|
| 退保编号 | 系统生成 |
| 保函编号 | 原保函编号 |
| 项目名称 | 对应项目 |
| 投标人 | 企业名称 |
| 退保金额 | 应退保费 |
| 退保原因 | 申请原因 |
| 申请时间 | 提交时间 |
| 状态 | 待审核/处理中/已通过/已拒绝 |
| 操作 | 审核、处理、查看 |

#### 3.4.3 状态说明

| 状态 | 说明 |
|------|------|
| 待审核 | 等待审核 |
| 处理中 | 正在处理退保 |
| 已通过 | 退保已执行 |
| 已拒绝 | 退保被拒绝 |

#### 3.4.4 退保审核

**审核要点**：
- 核对保函状态
- 确认是否满足退保条件
- 检查是否有未了结的理赔

**审核操作**：
- 通过 → 执行退保：注销保函、释放额度、原路退款
- 拒绝 → 填写拒绝原因，通知投标人

### 3.6 风控管理

#### 3.6.1 功能描述

配置风控规则，管理黑名单。

#### 3.6.2 功能模块

| 模块 | 功能点 |
|------|--------|
| 风控规则 | 规则配置、规则启停 |
| 黑名单 | 名单管理、添加/移除 |
| 审核配置 | 自动/人工审核设置 |

#### 3.6.3 风控规则

| 字段 | 说明 |
|------|------|
| 规则名称 | 规则描述 |
| 规则类型 | 金额限制、企业类型、地区限制 |
| 规则条件 | 条件表达式 |
| 动作 | 拦截/预警/放行 |
| 优先级 | 规则执行顺序 |
| 状态 | 启用/停用 |

**规则类型示例**：
- 单笔金额限制：> 80万 → 需人工审核
- 企业类型限制：禁止特定类型企业
- 地区限制：禁止特定地区
- 黑名单命中 → 直接拒绝

#### 3.6.4 黑名单管理

| 字段 | 说明 |
|------|------|
| 企业名称 | 企业全称 |
| 统一社会信用代码 | 18位代码 |
| 原因 | 加入原因 |
| 加入时间 | 创建时间 |
| 操作 | 查看详情、移除 |
