## 系统一：投标人门户（Portal）

### 核心功能模块

| 模块 | 功能点 |
|------|--------|
| 首页 | 项目列表、快捷入口、通知公告 |
| 项目大厅 | 项目搜索、详情查看、可投保证 |
| 投保申请 | 填写投保信息、选择金融机构 |
| 我的保函 | 保函列表、详情、下载 |
| 订单管理 | 订单列表、支付、退保申请 |
| 发票管理 | 发票申请、查看、下载 |
| 个人中心 | 实名认证、账户安全、消息通知 |

### 用户角色
- 投标人（企业）
- 经办人（代操作）

---

## 系统二：担保公司管理后台（Admin）

### 核心功能模块

| 模块 | 功能点 |
|------|--------|
| 工作台 | 待办事项、关键指标、快捷操作 |
| 投保管理 | 投保列表、投保审核、详情 |
| 保函管理 | 保函列表、签章、发放 |
| 退保管理 | 退保申请、审核、处理 |
| 机构管理 | 额度配置、费率设置 |
| 风控管理 | 黑名单、风控规则、审核配置 |
| 财务报表 | 收入统计、对账管理 |
| 系统管理 | 用户管理、角色权限 |

### 用户角色
- 风控岗
- 出函岗
- 财务岗
- 管理岗
