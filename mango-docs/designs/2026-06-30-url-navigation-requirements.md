## 1. 需求范围

本次建设“网址导航”能力，解决公司常用系统、公开链接、部门可见链接、指定用户可见链接和个人网址分散、难找、无法统一维护的问题。

本次包含：

- 用户侧菜单：`网址导航 / 公司网址`。
- 用户侧菜单：`网址导航 / 我的收藏`。
- 用户侧菜单：`网址导航 / 我的网址`。
- 后台菜单：`平台能力 / 网址管理 / 网址分类`。
- 后台菜单：`平台能力 / 网址管理 / 网址列表`。
- 网址可见范围：公开、公司内、指定部门、指定用户、个人。
- 网址分类：一级分类，不做多级 tree。
- 登录用户收藏可见范围内的网址。
- 登录用户登记、编辑、删除自己的个人网址。
- 提供 Open API 读取公开网址，供未登录页面、外部门户或第三方页面只读访问。

## 2. 不处理范围

- 不做小组对象、小组成员管理、小组网址管理。
- 不做多级分类树。
- 不做浏览器插件。
- 不做网页快照、离线归档、内容抓取。
- 不做网址访问统计。
- 不做网址有效性巡检。
- 不做 AI 自动分类。
- 不做公开注册链接；公开仅表示未登录也能查看公开网址。
- 不允许后台默认查看或管理用户个人网址；如需审计，必须另立需求。

## 3. 业务术语

| 术语 | 含义 |
|---|---|
| 网址导航 | 用户查找、打开、收藏网址的入口。 |
| 公司网址 | 用户侧展示的非个人网址，包括公开、公司内、指定部门、指定用户可见的网址。 |
| 我的收藏 | 登录用户收藏过的网址集合。 |
| 我的网址 | 登录用户自己登记、仅本人可见的个人网址。 |
| 网址分类 | 后台维护的一级分类，用于用户侧分组展示和搜索过滤。 |
| 网址列表 | 后台维护公开、公司内、指定部门、指定用户网址的管理页面。 |
| 可见范围 | 决定网址对哪些用户可见的业务规则。 |

### 4.1 网址属性

网址属性分为用户可见属性、展示控制属性、可见范围属性和后台维护属性。

| 属性 | 业务含义 | 适用范围 | 是否必填 | 用户侧是否展示 | Open API 是否返回 | 规则 |
|---|---|---|---|---|---|---|
| 网址名称 | 用户识别网址的名称 | 全部网址 | 是 | 是 | 是 | 不允许为空 |
| 网址地址 | 点击后打开的 URL | 全部网址 | 是 | 是 | 是 | 必须是合法 URL |
| 网址分类 | 网址所属一级分类 | 全部网址 | 公司网址必填，个人网址可选 | 是 | 是 | 只能选择启用分类 |
| 图标 | 网址卡片显示图标 | 全部网址 | 否 | 是 | 是 | 无图标时显示默认图标 |
| 简介 | 说明网址用途 | 全部网址 | 否 | 是 | 是 | 一句话说明 |
| 标签 | 用于搜索和识别 | 全部网址 | 否 | 是 | 是 | 多个标签按独立标签展示 |
| 可见范围 | 谁可以看到网址 | 公司网址、个人网址 | 是 | 后台展示，用户侧不展示 | 否 | 公开、公司内、指定部门、指定用户、个人 |
| 指定部门 | 指定部门可见时的部门列表 | 指定部门网址 | 条件必填 | 否 | 否 | 可见范围为指定部门时至少选择一个部门 |
| 指定用户 | 指定用户可见时的用户列表 | 指定用户网址 | 条件必填 | 否 | 否 | 可见范围为指定用户时至少选择一个用户 |
| 打开方式 | 点击网址后的打开方式 | 全部网址 | 是 | 否 | 是 | 本次固定新窗口打开 |
| 是否推荐 | 是否优先展示 | 公司网址 | 否 | 通过排序体现 | 是 | 推荐网址在同分类内靠前 |
| 排序号 | 展示排序 | 公司网址、分类 | 否 | 否 | 是 | 数字越小越靠前 |
| 状态 | 是否在用户侧展示 | 公司网址、分类 | 是 | 否 | 否 | 停用后用户侧和 Open API 都不返回 |
| 备注 | 后台维护说明 | 公司网址、分类 | 否 | 否 | 否 | 仅后台可见 |
| 创建人 | 记录创建来源 | 全部网址 | 系统生成 | 否 | 否 | 个人网址创建人决定可见性 |
| 更新时间 | 最近维护时间 | 全部网址 | 系统生成 | 后台展示 | 否 | 后台列表展示 |

| 对象ID | 唯一识别口径 | 归属口径 | 数量/有效期边界 | 重复判断口径 | 统计口径 |
|---|---|---|---|---|---|
| BO-001 | 同一租户下分类名称唯一 | 当前租户 | 无有效期 | 同一租户分类名称重复 | 本次不统计 |
| BO-002 | 同一租户下网址名称不要求唯一，网址地址可重复但需提示 | 当前租户 | 无有效期 | 同一租户同一 URL 已存在启用网址时提示 | 本次不统计 |
| BO-003 | 同一用户下网址名称不要求唯一，网址地址可重复但需提示 | 创建人 | 无有效期 | 同一用户同一 URL 已存在时提示 | 本次不统计 |
| BO-004 | 同一用户对同一网址只能收藏一次 | 收藏人 | 无有效期 | 用户和网址重复 | 本次不统计 |

## 6. 业务规则

| 规则ID | 规则 | 判断口径 | 允许结果 | 禁止/失败结果 | 关联功能 |
|---|---|---|---|---|---|
| BR-001 | 公开网址不要求登录 | 可见范围为公开，状态启用，分类启用 | 游客和登录用户可见 | 停用网址或停用分类不可见 | BF-001 |
| BR-002 | 公司内网址要求登录且属于当前租户 | 用户已登录，且属于当前租户 | 用户可见 | 未登录不可见 | BF-001 |
| BR-003 | 指定部门网址只对指定部门成员可见 | 用户已登录，用户部门命中网址配置部门 | 用户可见 | 非指定部门成员不可见 | BF-001 |
| BR-004 | 指定用户网址只对指定用户可见 | 用户已登录，用户命中网址配置用户 | 用户可见 | 非指定用户不可见 | BF-001 |
| BR-005 | 个人网址只对创建人可见 | 用户已登录，创建人等于当前用户 | 创建人可见 | 其他用户不可见 | BF-003 |
| BR-006 | 停用网址不在用户侧展示 | 网址状态为停用 | 后台可查 | 用户侧不可见 | BF-001 |
| BR-007 | 停用分类不在用户侧展示 | 分类状态为停用 | 后台可查 | 分类及分类下网址用户侧不可见 | BF-001 |
| BR-008 | 收藏不能重复 | 同一用户已收藏同一网址 | 返回已收藏状态 | 不新增重复收藏 | BF-002 |
| BR-009 | 已不可见网址不出现在我的收藏 | 用户不再满足网址可见范围 | 我的收藏不展示该网址 | 不展示“失效占位” | BF-002 |
| BR-010 | 分类下存在启用网址时不可删除分类 | 分类下存在启用网址 | 可停用分类 | 删除失败并提示先停用或迁移网址 | BF-004 |
| BR-011 | 后台网址列表不管理个人网址 | 可见范围为个人 | 仅本人在我的网址维护 | 后台网址列表不展示个人网址 | BF-005 |
| BR-012 | Open API 只返回公开网址 | 可见范围为公开，网址启用，分类启用，租户上下文明确 | 返回公开网址 | 不返回公司内、指定部门、指定用户、个人网址 | BF-006 |
| BR-013 | Open API 不返回后台维护字段 | Open API 响应字段 | 返回名称、URL、分类、图标、简介、标签、打开方式、排序 | 不返回备注、状态、创建人、指定部门、指定用户 | BF-006 |

## 8. 页面原型需求

### 8.1 PG-001 公司网址

#### 什么情况下

| 前提 | 规则 |
|---|---|
| 未登录 | 只能看到公开且启用的网址 |
| 已登录 | 可以看到公开、公司内、指定部门、指定用户范围内启用的网址 |
| 分类停用 | 分类和分类下网址不展示 |

#### 什么角色

| 角色 | 可见数据 | 可执行操作 | 不可执行操作 |
|---|---|---|---|
| 游客 | 公开网址 | 搜索、打开 | 收藏、查看公司内/部门/指定用户网址 |
| 登录用户 | 自己可见范围内的网址 | 搜索、打开、收藏、取消收藏 | 查看无权限网址 |

#### 在哪里

| 入口 | 操作路径 | 打开方式 |
|---|---|---|
| 用户菜单 | 网址导航 / 公司网址 | 当前页打开 |
| 网址卡片 | 网址导航 / 公司网址 / 网址卡片 | 新窗口打开目标网址 |

#### 做什么事情

| 项 | 内容 |
|---|---|
| 业务动作 | 查看和打开公司网址 |
| 业务结果 | 用户找到并打开自己可见的网址 |
| 影响对象 | BO-002 网址、BO-004 收藏记录 |
| 本功能不做 | 不维护网址、不维护分类、不新增个人网址 |

#### 怎么做

页面布局：

| 区域 | 内容 | 规则 |
|---|---|---|
| 搜索区 | 关键词、分类 | 关键词匹配名称、简介、标签、URL |
| 分类区 | 启用分类 | 一级分类横向或侧边展示，不展示 tree |
| 网址区 | 网址卡片 | 按分类、推荐、排序号展示 |
| 卡片操作 | 打开、收藏/取消收藏 | 收藏按钮仅登录用户可用 |

列表或卡片字段：

| 字段 | 显示格式 | 来源 |
|---|---|---|
| 网址名称 | 文本 | 网址 |
| 图标 | 图标图片；无图标时显示默认图标 | 网址 |
| 简介 | 一句话说明 | 网址 |
| 标签 | 标签文案 | 网址 |
| 分类 | 分类名称 | 网址分类 |

交互步骤：

1. 用户进入“网址导航 / 公司网址”。
2. 系统按登录状态和可见范围加载网址。
3. 用户输入关键词或选择分类。
4. 系统刷新网址卡片。
5. 用户点击网址卡片，系统新窗口打开目标网址。
6. 登录用户点击收藏，系统将网址加入“我的收藏”。
7. 未登录用户点击收藏，系统提示登录后收藏。

#### 如何确定做的是对的

| 验收ID | 场景 | 什么情况下 | 什么角色 | 操作 | 期望用户可见结果 |
|---|---|---|---|---|---|
| AC-001 | 未登录查看公开网址 | 存在公开启用网址 | 游客 | 打开公司网址页 | 只看到公开网址 |
| AC-002 | 登录查看公司内网址 | 存在公司内启用网址 | 登录用户 | 打开公司网址页 | 看到公开和公司内网址 |
| AC-003 | 部门网址隔离 | 网址指定部门 A | 部门 B 用户 | 打开公司网址页 | 看不到部门 A 网址 |
| AC-004 | 搜索网址 | 存在名称或标签命中的网址 | 登录用户 | 输入关键词 | 只展示命中且有权查看的网址 |
| AC-005 | 收藏网址 | 用户已登录且网址可见 | 登录用户 | 点击收藏 | 收藏状态变为已收藏，并出现在我的收藏 |

### 8.2 PG-002 我的收藏

#### 什么情况下

| 前提 | 规则 |
|---|---|
| 用户已登录 | 可以进入我的收藏 |
| 用户未登录 | 跳转登录或提示登录后查看 |

#### 什么角色

| 角色 | 可见数据 | 可执行操作 | 不可执行操作 |
|---|---|---|---|
| 登录用户 | 本人收藏且当前仍可见的网址 | 打开、取消收藏 | 查看他人收藏 |

#### 在哪里

| 入口 | 操作路径 | 打开方式 |
|---|---|---|
| 用户菜单 | 网址导航 / 我的收藏 | 当前页打开 |

#### 做什么事情

| 项 | 内容 |
|---|---|
| 业务动作 | 查看和管理收藏的网址 |
| 业务结果 | 用户快速打开常用网址 |
| 影响对象 | BO-004 收藏记录 |
| 本功能不做 | 不新增公司网址、不新增个人网址 |

#### 怎么做

1. 用户进入“网址导航 / 我的收藏”。
2. 系统加载本人收藏且当前仍可见的网址。
3. 用户点击网址，系统新窗口打开目标网址。
4. 用户点击取消收藏，系统移除收藏记录。
5. 收藏网址因停用、分类停用或权限变化不可见时，本页面不展示该网址。

#### 如何确定做的是对的

| 验收ID | 场景 | 什么情况下 | 什么角色 | 操作 | 期望用户可见结果 |
|---|---|---|---|---|---|
| AC-006 | 查看收藏 | 用户已收藏网址 | 登录用户 | 打开我的收藏 | 展示本人收藏的网址 |
| AC-007 | 取消收藏 | 用户已收藏网址 | 登录用户 | 点击取消收藏 | 网址从我的收藏移除 |
| AC-008 | 不展示无权限收藏 | 收藏网址已不可见 | 登录用户 | 打开我的收藏 | 不展示该网址 |

#### 什么角色

|---|---|---|---|
| 外部页面 | 公开、启用分类、启用状态的网址 | 查询、展示、打开 | 收藏、查看公司内/部门/指定用户/个人网址 |

#### 在哪里

| 入口 | 操作路径 | 打开方式 |
|---|---|---|

#### 做什么事情

| 项 | 内容 |
|---|---|
| 业务动作 | 查询公开网址 |
| 业务结果 | 外部页面只拿到公开网址 |
| 影响对象 | BO-002 网址 |

#### 怎么做

1. 调用方请求公开网址 Open API，并传入租户上下文。
2. 系统校验租户上下文。

4. 系统按分类、推荐、排序号返回。
5. 系统响应中不包含后台维护字段和可见目标字段。

响应字段：

| 字段 | 说明 |
|---|---|
| 网址名称 | 用户展示名称 |
| 网址地址 | 点击打开 URL |
| 分类 | 分类名称 |
| 图标 | 图标地址或默认图标标识 |
| 简介 | 网址用途 |
| 标签 | 标签列表 |
| 打开方式 | 新窗口 |
| 排序号 | 展示排序 |

#### 如何确定做的是对的

| 验收ID | 场景 | 什么情况下 | 什么角色 | 操作 | 期望用户可见结果 |
