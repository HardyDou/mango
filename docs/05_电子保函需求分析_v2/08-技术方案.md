## 8.2 通道适配器设计

### 8.2.1 适配器模式

```python
# 适配器接口定义
class GuaranteeChannelAdapter:
    def apply_for_guarantee(self, apply_info): ...
    def issue_guarantee(self, issue_info): ...
    def verify_guarantee(self, guarantee_no): ...
    def cancel_guarantee(self, cancel_info): ...

# 具体通道适配器
class HunanChannelAdapter(GuaranteeChannelAdapter): pass
class AnhuiChannelAdapter(GuaranteeChannelAdapter): pass
class BankDirectAdapter(GuaranteeChannelAdapter): pass
```

### 8.2.2 平台配置设计

```
每个上游平台需独立配置：
- platformCode: 平台编码
- signAlgorithm: 签名算法 (SM3/MD5/SM2)
- signParams: 参与签名的参数列表
- encryptAlgorithm: 加密算法
- secretKey: appSecret
- caCertPath: CA证书路径
- ipWhitelist: IP白名单
- protocolType: 协议类型 (HTTP/SDK/MQ/FILE)
```

## 8.3 接口安全机制

### 8.3.1 接口防篡改

| 平台类型 | 签名算法 | 密钥 | 典型平台 |
|----------|----------|------|----------|
| 国密SM3 | SM3 | appkey + appsecret | 数字泸州、眉山4.0 |
| MD5 | MD5 | appkey + appsecret | 蔓延-见费模式 |
| SM2签名 | SM2 | CA证书 | 部分标准规范 |

### 8.3.2 业务数据加密

| 方案 | 加密内容 | 密钥来源 | 典型平台 |
|------|----------|----------|----------|
| 数字信封模式 | applydata (SM4) + digitalenvelope | 投标人CA证书公钥 | 数字泸州 |
| 平台密钥模式 | 整个请求体用appSecret SM4加密 | 平台分配appSecret | 江西省、眉山 |

### 8.3.3 签名流程（SM3示例）

```
1. 参数按ASCII升序排序
2. 拼接成 "key=value&key=value" 格式
3. 末尾拼接 appsecret
4. 计算SM3值作为sign
```

## 8.16 核心数据库表设计

### 8.16.1 通道配置表

```sql
CREATE TABLE channel_config (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    channel_code VARCHAR(50) NOT NULL,
    channel_name VARCHAR(100),
    channel_type VARCHAR(20),
    sign_algorithm VARCHAR(20),
    encrypt_algorithm VARCHAR(20),
    secret_key VARCHAR(500),
    api_url VARCHAR(500),
    status VARCHAR(20) DEFAULT 'ACTIVE',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

### 8.16.2 项目信息表

```sql
CREATE TABLE project_info (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    project_no VARCHAR(100) NOT NULL,
    project_name VARCHAR(500),
    tenderno VARCHAR(100),
    bidbond_amount DECIMAL(18,2),
    deadline DATETIME,
    channel_code VARCHAR(50),
    extend_data TEXT,
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP
);
```
