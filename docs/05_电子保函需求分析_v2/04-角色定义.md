## 4.2 外部用户

### 4.2.1 投标人

| 属性 | 说明 |
|------|------|
| **定义** | 参与投标的企业或个人 |
| **来源** | 从交易系统同步或自行注册 |
| **权限** | 投保申请、保函查询、退保申请、发票下载 |

### 4.2.2 招标人

| 属性 | 说明 |
|------|------|
| **定义** | 发布招标项目的单位 |
| **来源** | 从交易系统同步 |
| **权限** | 保函验真、保函查询 |

### 4.2.3 经办人

| 属性 | 说明 |
|------|------|
| **定义** | 投标人方办理保函人员 |
| **来源** | 投标人添加 |
| **权限** | 代办投保、代办查询、代办支付 |

### 4.2.4 受益人

| 属性 | 说明 |
|------|------|
| **定义** | 保函赔付对象，通常为招标人 |
| **来源** | 从交易系统同步 |
| **权限** | 保函查询、验真（需授权） |

## 4.3 机构用户

### 4.3.1 风控岗

| 属性 | 说明 |
|------|------|
| **定义** | 金融机构风控审核人员 |
| **权限** | 风控审核、风控规则配置、黑名单管理 |

### 4.3.2 出函岗

| 属性 | 说明 |
|------|------|
| **定义** | 金融机构出函操作人员 |
| **权限** | 保函开具、保函查询、退保处理 |

### 4.3.3 财务岗

| 属性 | 说明 |
|------|------|
| **定义** | 金融机构财务人员 |
| **权限** | 发票管理、对账查询、报表查看 |

### 4.3.4 管理岗

| 属性 | 说明 |
|------|------|
| **定义** | 金融机构管理人员 |
| **权限** | 机构配置、用户管理、报表查看 |

---

## 4.4 运营用户

### 4.4.1 运营岗

| 属性 | 说明 |
|------|------|
| **定义** | 平台运营人员 |
| **权限** | 通道管理、机构管理、订单管理、通知管理 |

### 4.4.2 财务岗

| 属性 | 说明 |
|------|------|
| **定义** | 平台财务人员 |
| **权限** | 发票管理、对账管理、账务查询 |

### 4.4.3 风控岗

| 属性 | 说明 |
|------|------|
| **定义** | 平台风控人员 |
| **权限** | 风控规则配置、黑名单管理、风险监控 |

### 4.4.4 管理员

| 属性 | 说明 |
|------|------|
| **定义** | 系统管理员 |
| **权限** | 系统配置、用户管理、角色管理、全局查看 |

## 4.6 数据权限

### 4.6.1 投标人数据权限

- 只能查看自己作为投标人的保函
- 只能查看自己作为经办人的保函

### 4.6.2 招标人数据权限

- 只能查看所属项目的保函
- 只能验真所属项目的保函

### 4.6.3 机构数据权限

- 只能查看本机构的保函
- 按机构配置的数据范围

### 4.6.4 运营数据权限

- 按通道划分数据范围
- 按租户划分数据范围
