# 02-术语表

本文档定义电子保函系统中使用的专业术语。

---

## 2.1 业务术语

| 术语 | 英文 | 说明 |
|------|------|------|
| **电子保函** | Electronic Guarantee | 通过电子方式开具的保函，具有法律效力 |
| **投标保证金保函** | Bid Bond Guarantee | 投标阶段替代保证金的保函 |
| **履约保证金保函** | Performance Bond | 中标后为保障合同履行开具的保函 |
| **工程质量保证金保函** | Quality Bond | 保障工程质量的保函 |
| **农民工工资支付保函** | Wage Guarantee | 保障农民工工资支付的保函 |
| **分离式银行保函** | Separate Bank Guarantee | 担保公司担保，银行出具 |
| **保证保险** | Surety Insurance | 保险公司出具的保险单 |
| **见费模式** | Pay Before Issue | 保费支付后才出具保函 |
| **后费模式** | Pay After Issue | 出函后再支付保费 |
| **批量出函** | Batch Issue | 同一标段多家投标人同时出函 |
| **批量验真** | Batch Verify | 开标时批量验证多家投标人保函 |

## 2.2 技术术语

| 术语 | 英文 | 说明 |
|------|------|------|
| **SM2** | SM2 | 国密非对称加密算法 |
| **SM3** | SM3 | 国密哈希算法 |
| **SM4** | SM4 | 国密对称加密算法 |
| **数字信封** | Digital Envelope | SM2加密SM4密钥的方案 |
| **CA证书** | CA Certificate | 数字证书认证机构颁发 |
| **电子签章** | Electronic Signature | 文档的数字签名 |
| **OFD** | Open Fixed-layout Document | 国产文档格式 |
| **PDF** | Portable Document Format | 便携式文档格式 |
| **适配器模式** | Adapter Pattern | 处理不同接口标准的模式 |

## 2.3 角色术语

| 术语 | 英文 | 说明 |
|------|------|------|
| **投标人** | Bidder | 参与投标的企业或个人 |
| **招标人** | Tenderee | 发布招标项目的单位 |
| **受益人** | Beneficiary | 保函赔付对象 |
| **出函机构** | Issuing Institution | 银行/保险公司/担保公司 |
| **通道** | Channel | 上游交易系统接入渠道 |
| **经办人** | Operator | 投标人方办理保函人员 |

---

## 2.4 流程术语

| 术语 | 英文 | 说明 |
|------|------|------|
| **投保申请** | Application | 投标人提交保函申请 |
| **保函开具** | Issue | 出函机构出具保函 |
| **保函验真** | Verification | 验证保函真伪 |
| **退保** | Cancellation | 申请人主动注销保函 |
| **注销** | Release | 保函到期或条件满足后注销 |
| **理赔** | Claim | 触发赔付条件后的处理 |
