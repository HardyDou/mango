## 2. 外部用户

### 2.1 投标人

| 属性 | 说明 |
|------|------|
| **定义** | 参与投标的企业或个人 |
| **身份** | 企业：统一社会信用代码<br>个人：身份证 |
| **操作** | 投保申请、支付保费、查看保函、申请退保 |
| **入口** | 通过交易系统跳转 |

### 2.2 招标人

| 属性 | 说明 |
|------|------|
| **定义** | 发布招标项目的单位 |
| **身份** | 统一社会信用代码 |
| **操作** | 项目信息推送、保函验真 |
| **入口** | 交易系统 |

### 2.3 招标代理机构

| 属性 | 说明 |
|------|------|
| **定义** | 代理招标业务的机构 |
| **身份** | 统一社会信用代码 |
| **操作** | 项目管理、保函查询 |
| **入口** | 交易系统 |

### 2.4 经办人

| 属性 | 说明 |
|------|------|
| **定义** | 企业具体办理业务的人员 |
| **身份** | 个人身份证 + 企业授权 |
| **操作** | 发起投保、代办支付 |
| **入口** | 交易系统/保函平台 |

## 3. 内部用户

### 3.1 运营人员

| 属性 | 说明 |
|------|------|
| **职责** | 日常运营管理 |
| **权限** | 订单管理、用户管理、机构管理 |
| **操作** | 订单处理、用户审核、机构配置 |

### 3.2 风控人员

| 属性 | 说明 |
|------|------|
| **职责** | 风险控制 |
| **权限** | 风控规则配置、黑名单管理 |
| **操作** | 风险审核、规则调整 |

### 3.3 财务人员

| 属性 | 说明 |
|------|------|
| **职责** | 财务管理 |
| **权限** | 账单查看、对账管理 |
| **操作** | 财务报表、账务核对 |

### 3.4 管理员

| 属性 | 说明 |
|------|------|
| **职责** | 系统管理 |
| **权限** | 全功能 |
| **操作** | 系统配置、权限管理 |

---

## 4. 机构用户

| 属性 | 说明 |
|------|------|
| **定义** | 银行方业务办理人员 |
| **机构** | 接入的银行 |
| **操作** | 保函审核、出函处理 |

### 4.2 保险经理

| 属性 | 说明 |
|------|------|
| **定义** | 保险公司业务人员 |
| **机构** | 接入的保险公司 |
| **操作** | 保单审核、出单处理 |

### 4.3 担保经理

| 属性 | 说明 |
|------|------|
| **定义** | 担保公司业务人员 |
| **机构** | 接入的担保公司 |
| **操作** | 担保审核、出函处理 |
