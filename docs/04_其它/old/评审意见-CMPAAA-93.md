## 补充内容确认

### 1. 保费计算规则 ✅ 已完善

| 补充项 | 要求 | 实际内容 | 状态 |
|--------|------|----------|------|
| 有效期计算方式 | 明确计算方式 | 按实际天数/365转换为年，不足整年按整年计算 | ✅ |
| 金额精度处理 | 明确舍入规则 | 保留两位小数，第三位四舍五入，最终金额向上取整到元 | ✅ |
| 最低保费 | 明确具体规则 | 不低于200元，若计算结果低于200元，按200元收取 | ✅ |
| 阶梯费率 | 明确具体费率区间 | 10万（含）以下：2.0%-3.0%，10万-50万：1.5%-2.5%，50万-100万：1.0%-2.0%，100万-500万：0.8%-1.5%，500万以上：0.5%-1.0% | ✅ |

**保费计算示例**：
- 示例1：30万，费率1.8%，1年 → 5400元 ✅
- 示例2：80万，费率1.2%，6个月（按1年）→ 9600元 ✅
- 示例3：5万，费率2.5%，1年 → 1250元（>200元） ✅

### 2. 状态流转 ✅ 已完善

| 补充项 | 要求 | 实际内容 | 状态 |
|--------|------|----------|------|
| PREMIUM_REFUSED状态 | 体现缴费失败状态 | 已添加到状态列表和流转图中 | ✅ |
| 已缴费撤销流程 | 明确退款审核流程 | 4步流程：提交撤销申请 → 机构管理员审核 → 财务确认 → 退款成功 | ✅ |
| REJECTED后重新提交 | 明确重新提交逻辑 | 保留原申请记录，生成新的申请版本 | ✅ |

**状态流转图**：
```
草稿(DRAFT) → 已提交(SUBMITTED) → 审核中(REVIEWING)
                                      ↓
                            审核通过(APPROVED) → 已缴费(PREMIUM_PAID) → 保函出具
                            ↓                         ↓
                      审核拒绝(REJECTED)          缴费失败(PREMIUM_REFUSED) → 重新缴费
                      ↓                                    ↓
                    重新提交 → 已提交(SUBMITTED)      已撤销(CANCELLED)
                                      ↓
                            已撤销(CANCELLED)
```
