## 一、模块概述

投保申请与审核模块是电子保函系统的核心功能模块，负责接收投标人的保函申请，进行资质审核，并计算保费。本模块连接机构管理与保函出具两个环节。

### 2.3 保费计算

| 功能 | 描述 | 优先级 | 负责人 |
|------|------|--------|--------|
| 保费计算 | 根据保额、类型、机构费率计算保费 | P0 | 后端开发工程师 |
| 费率查询 | 查询对应机构的保费费率 | P0 | 后端开发工程师 |
| 保费优惠 | 根据机构优惠政策计算优惠 | P1 | 后端开发工程师 |
| 保费试算 | 申请前预估保费金额 | P0 | 后端开发工程师 |

**保费计算规则**：
```
保费 = 保函金额 × 年费率 × 有效期(年)
```

#### 2.3.1 有效期计算方式
- **计算方式**：按实际天数/365转换为年，不足整年按整年计算
- **最小单位**：最小按1年计算，不足1年按1年收费
- **最大期限**：根据机构配置和招标文件要求，最长不超过3年

#### 2.3.2 金额精度处理
- **计算精度**：保费计算保留两位小数，第三位四舍五入
- **最终金额**：最终保费金额统一向上取整到元
- **最低保费**：不低于200元，若计算结果低于200元，按200元收取

#### 2.3.3 阶梯费率标准
| 保函金额 | 年费率区间 | 备注 |
|----------|------------|------|
| 10万（含）以下 | 2.0%-3.0% | 最低档位 |
| 10万-50万（含） | 1.5%-2.5% | 标准费率 |
| 50万-100万（含） | 1.0%-2.0% | 优惠费率 |
| 100万-500万（含） | 0.8%-1.5% | 大额优惠 |
| 500万以上 | 0.5%-1.0% | 协议费率，需审批 |

#### 2.3.4 保费计算示例
```
示例1：保函金额30万，费率1.8%，有效期1年
保费 = 300000 × 1.8% × 1 = 5400元

示例2：保函金额80万，费率1.2%，有效期6个月
有效期按1年计算 = 800000 × 1.2% × 1 = 9600元

示例3：保函金额5万，费率2.5%，有效期1年
保费 = 50000 × 2.5% × 1 = 1250元（>200元，正常计算）
```

## 三、状态流转

```
草稿(DRAFT) → 已提交(SUBMITTED) → 审核中(REVIEWING)
                                        ↓
                              审核通过(APPROVED) → 已缴费(PREMIUM_PAID) → 保函出具
                              ↓                         ↓
                        审核拒绝(REJECTED)          缴费失败(PREMIUM_REFUSED) → 重新缴费
                        ↓                                    ↓
                      重新提交 → 已提交(SUBMITTED)      已撤销(CANCELLED)
                                        ↓
                              已撤销(CANCELLED)
```

### 3.1 状态说明

| 状态 | 说明 | 可转换状态 |
|------|------|-----------|
| DRAFT | 草稿 | SUBMITTED, CANCELLED |
| SUBMITTED | 已提交（待审核） | REVIEWING, CANCELLED |
| REVIEWING | 审核中 | APPROVED, REJECTED |
| APPROVED | 审核通过 | PREMIUM_PAID, CANCELLED |
| REJECTED | 审核拒绝 | SUBMITTED（重新提交）, CANCELLED |
| PREMIUM_PAID | 已缴费 | 保函出具流程 |
| PREMIUM_REFUSED | 缴费失败 | PREMIUM_PAID（重新缴费）, CANCELLED |
| CANCELLED | 已撤销 | - |

### 3.2 状态流转业务规则

#### 3.2.1 审核拒绝后重新提交
- 审核拒绝后，申请人可修改申请材料后重新提交
- 重新提交后状态从 REJECTED 转为 SUBMITTED
- 重新提交保留原申请记录，并生成新的申请版本

#### 3.2.2 缴费失败处理
- 缴费失败后，系统自动进入 PREMIUM_REFUSED 状态
- 申请人可重新发起缴费
- 缴费失败超过3次后，系统自动锁定申请，需联系管理员处理
- 锁定后需管理员手动解锁才能继续缴费

#### 3.2.3 已缴费撤销流程（退款审核）
- 审核通过后至保函出具前，申请人可申请撤销
- 已缴费用户撤销需进行退款审核
- 退款审核流程：
  1. 申请人提交撤销申请
  2. 机构管理员审核退款
  3. 财务确认后原路退回
  4. 退款成功后状态变为 CANCELLED
