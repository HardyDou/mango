### 2.2 关键指标卡片

| 指标 | 标题 | 数值格式 | 趋势 |
|------|------|----------|------|
| 保函总量 | 保函总量 | 整数 | 同比 |
| 在保金额 | 在保金额 | 亿/万 | 同比 |
| 今日新增 | 今日新增 | 整数 | 环比 |
| 保费收入 | 保费收入(本月) | 亿/万 | 同比 |

### 2.3 指标卡片规范

```
+-----------------------+
|  标题              ↗  |  ← 14px, #606266, 右上趋势图标
+-----------------------+
|  数值                |  ← 28px, #1F1F1F, 字重700
+-----------------------+
|  同比/环比变化       |  ← 12px, #00A854(升)/#F5222D(降)
+-----------------------+
```

### 2.4 图表配色

| 图表类型 | 配色方案 |
|----------|----------|
| 保函数量趋势 | #0066FF (主色) |
| 保费收入趋势 | #00A854 (成功色) |
| 保函类型分布 | #0066FF, #00A854, #FA8C15, #F5222D |
| 机构排行 | 从深到浅蓝色渐变 |

## 八、图表规范

### 8.1 折线图

- 线条粗细: 2px
- 数据点: 4px 圆点
- 悬浮显示数值
- 支持区域填充(可选)

### 8.2 饼图/环形图

- 起始角度: 12点方向顺时针
- 悬浮显示百分比
- 中心可显示总量

### 8.3 柱状图

- 柱子间距: 20px
- 柱子圆角: 4px
- 悬浮显示数值

### 8.4 图表颜色

```
主色系: #0066FF
成功色: #00A854
警告色: #FA8C15
错误色: #F5222D
中性色: #909399

图表渐变:
- 深蓝: #0052CC
- 浅蓝: #0066FF
- 极浅蓝: #E6F0FF
```
