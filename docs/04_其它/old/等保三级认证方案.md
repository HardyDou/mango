## 1. 等保三级认证概述

### 1.1 什么是等保三级

等级保护是我国网络安全的基本制度，对信息系统按等级进行保护。等保三级是国家对非银行机构的最高级认证，属于"监督保护级"，适用于：
- 金融行业信息系统
- 医疗健康信息系统
- 政务服务系统
- 公共资源交易系统

电子保函系统涉及金融机构交易数据，**必须通过等保三级认证**。

### 1.2 等保三级技术要求

| 安全层面 | 要求项数 | 关键控制点 |
|---------|---------|-----------|
| 安全物理环境 | 10 | 机房选址、防盗、防火、防水、温湿度控制 |
| 安全通信网络 | 10 | 网络架构、边界防护、通信传输 |
| 安全区域边界 | 11 | 边界防护、访问控制、入侵防范 |
| 安全计算环境 | 23 | 身份鉴别、访问控制、安全审计、恶意代码防范 |
| 安全管理中心 | 5 | 系统管理、审计管理、安全管理 |
| 安全管理制度 | 7 | 管理制度、评审和修订 |
| 安全管理机构 | 7 | 岗位设置、人员配备、授权审批 |
| 安全管理人员 | 7 | 人员录用、离岗、安全意识 |
| 安全建设管理 | 12 | 系统定级、安全方案设计、产品采购 |
| 安全运维管理 | 14 | 环境管理、介质管理、设备维护 |

## 2. 测评机构选择

### 2.1 推荐测评机构

| 机构名称 | 资质 | 特点 | 费用参考 |
|---------|------|------|---------|
| **中国信息安全测评中心** | CNAS认可 | 权威、权威性强 | 15-25万 |
| **公安部第一研究所** | 等保测评机构 | 公安体系认可 | 12-20万 |
| **北京知道创宇** | 等保测评机构 | 速度快、报告专业 | 10-18万 |
| **启明星辰** | 等保测评机构 | 综合安全厂商 | 12-18万 |

**推荐选择**: 公安部第一研究所或启明星辰（性价比高、周期可控）

### 2.2 选择要点

1. **资质合规**: 必须持有省级公安部门颁发的《等级保护测评机构推荐证书》
2. **经验丰富**: 有金融/政务行业等保案例
3. **本地化服务**: 便于现场测评和沟通
4. **售后服务**: 整改支持、复测配合

## 3. 认证流程

### 3.1 整体流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  系统定级   │───▶│  备案       │───▶│  建设整改  │───▶│  测评       │───▶│  监督检查  │
│  (专家评审) │    │  (公安)    │    │  (自查)    │    │  (第三方)   │    │  (公安)    │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
      │                  │                  │                  │                  │
     1-2周             1周              2-4周             1-2周              持续
```

### 3.2 详细步骤

#### Step 1: 系统定级（1-2周）
- 编写《系统安全等级保护定级报告》
- 专家评审会（3名以上专家）
- 报送公安网安部门备案

#### Step 2: 备案（1周）
- 提交备案材料至所在地公安局
- 获取《信息系统安全等级保护备案证明》

#### Step 3: 建设整改（2-4周）
- 差距分析（初测）
- 安全加固
- 管理制度完善

#### Step 4: 测评（1-2周）
- 正式测评（技术+管理）
- 报告编制
- 整改复核

#### Step 5: 监督检查（持续）
- 公安定期检查
- 年度复测
