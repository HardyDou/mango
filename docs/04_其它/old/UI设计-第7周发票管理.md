### 2.2 表格列定义

| 列名 | 宽度 | 内容 |

| 销售方 | 180px | 出函机构名称 |

| 税额 | 100px | 税额，右对齐 |
| 价税合计 | 120px | 含税金额，右对齐，加粗 |

| 保函编号 | 140px | 关联保函 |

## 三、发票开具页面

### 3.1 页面结构

```
+--------------------------------------------------+
|  顶部导航栏                                       |
+--------+-----------------------------------------+
| 侧边栏  |  内容区域                              |
|        |  +----------------------------------+  |
|        |  | 页面标题: 开具发票               |  |
|        |  +----------------------------------+  |
|        |  | 步骤条:                          |  |
|        |  | [1]选择保函>[2]填写信息>[3]确认 |  |
|        |  +----------------------------------+  |
|        |  |                                  |  |
|        |  | 步骤1: 选择保函                 |  |
|        |  | 可开票保函列表                    |  |
|        |  | □ | 保函编号 | 投保人 | 保费   |  |
|        |  |   |        |        |        |  |
|        |  +----------------------------------+  |
|        |  |                                  |  |
|        |  | 步骤2: 发票信息                  |  |
|        |  | +------------------------------+|  |
|        |  | | 发票类型: ○专用 ○普通 ○电子  ||  |
|        |  | +------------------------------+|  |
|        |  | | 购买方信息:                   ||  |
|        |  | | [名称] [税号]                ||  |
|        |  | | [地址] [电话]                ||  |
|        |  | | [银行] [账号]                ||  |
|        |  | +------------------------------+|  |
|        |  | | 销售方信息: (自动填充)        ||  |
|        |  | +------------------------------+|  |
|        |  | | 发票明细:                     ||  |
|        |  | | 货物名称 | 金额 | 税率 | 税额 ||  |
|        |  | | 担保服务费| 1000 | 6% | 60  ||  |
|        |  | +------------------------------+|  |
|        |  | | 价税合计: ¥1,060.00          ||  |
|        |  | +------------------------------+|  |
|        |  | | 交付方式: ○下载 ○邮箱 ○邮寄  ||  |
|        |  +----------------------------------+  |
|        |  |                                  |  |
|        |  | 步骤3: 确认提交                  |  |
|        |  | 发票信息摘要卡片                 |  |
|        |  +----------------------------------+  |
|        |  | 底部操作栏: [取消] [提交开具]   |  |
|        |  +----------------------------------+  |
+--------+-----------------------------------------+
```

### 3.2 发票类型与字段

| 发票类型 | 必填字段 |
|----------|----------|
| 增值税专用发票 | 名称、税号、地址、电话、银行、账号 |
| 增值税普通发票 | 名称、税号 |
| 电子发票 | 名称、税号、邮箱 |

### 3.3 金额计算

| 字段 | 计算规则 |
|------|----------|
| 金额 | 用户输入（不含税） |
| 税率 | 固定6% |
| 税额 | 金额 × 税率 |
| 价税合计 | 金额 + 税额 |

## 五、发票红冲页面

### 5.1 页面结构

```
+--------------------------------------------------+
|  顶部导航栏                                       |
+--------+-----------------------------------------+
| 侧边栏  |  内容区域                              |
|        |  +----------------------------------+  |
|        |  | 页面标题: 发票红冲               |  |
|        |  +----------------------------------+  |
|        |  |                                  |  |
|        |  | 原发票信息卡片                   |  |
|        |  | - 发票号码: 1234567890          |  |
|        |  | - 发票类型: 增值税专用发票       |  |
|        |  | - 价税合计: ¥1,060.00           |  |
|        |  | - 开票日期: 2026-03-10          |  |
|        |  | - 已开票天数: 3天               |  |
|        |  +----------------------------------+  |
|        |  |                                  |  |
|        |  | 红冲申请表单                     |  |
|        |  | 红冲类型:                        |  |

|        |  |                                  |  |

|        |  |                                  |  |
|        |  | 红冲原因: [下拉选择]            |  |
|        |  | - 退保退款                       |  |
|        |  | - 开票信息错误                   |  |
|        |  | - 金额调整                       |  |
|        |  | - 其他                           |  |
|        |  |                                  |  |
|        |  | 补充说明: [文本域]               |  |
|        |  +----------------------------------+  |
|        |  | 风险提示:                        |  |

|        |  | - 红冲操作不可撤销               |  |
|        |  +----------------------------------+  |
|        |  | 底部操作栏: [取消] [确认红冲]   |  |
|        |  +----------------------------------+  |
+--------+-----------------------------------------+
```

### 5.2 红冲类型选择

| 类型 | 说明 | 后续操作 |
|------|------|----------|
| 全额红冲 | 红冲全部金额 | 可直接完成 |
| 部分红冲 | 红冲部分金额 | 需重新开具发票 |

### 5.3 红冲确认

- 点击确认红冲后，弹出二次确认对话框
- 确认对话框显示：
  - 原发票信息
  - 红冲金额
  - 红冲原因
