## 一、设计原则

### 1.1 核心原则

| 原则 | 描述 |
|------|------|
| **专业可靠** | 金融行业属性决定设计必须严谨、专业、可信 |
| **高效便捷** | 减少用户操作步骤，提升业务办理效率 |
| **清晰一致** | 统一的视觉语言，降低学习成本 |
| **安全合规** | 符合金融行业安全标准和合规要求 |

### 1.2 设计目标

- **可用性**: 关键业务流程3步内完成
- **可访问性**: 符合 WCAG 2.1 AA 标准
- **响应式**: 支持 1920px、1440px、1280px 等常见分辨率
- **性能**: 首屏加载时间 < 2秒

## 五、组件规范

### 5.1 按钮 (Button)

| 类型 | 样式 | 用途 |
|------|------|------|
| 主按钮 | 蓝色填充 `#0066FF`，圆角 4px | 主要操作：提交、确认 |
| 次按钮 | 白色填充，边框 `#DCDFE6` | 次要操作：取消、返回 |
| 文字按钮 | 无背景，无边框 | 辅助操作：更多选项 |
| 危险按钮 | 红色填充 `#F5222D` | 危险操作：删除、退保 |

**尺寸**:
- 大按钮: 高度 40px，文字 16px
- 中按钮: 高度 36px，文字 14px
- 小按钮: 高度 32px，文字 12px

### 5.2 输入框 (Input)

- 高度: 36px (默认), 32px (紧凑)
- 边框: 1px solid `#DCDFE6`
- 圆角: 4px
- 聚焦边框: 1px solid `#0066FF`
- 错误边框: 1px solid `#F5222D`

### 5.3 表格 (Table)

- 表头背景: `#F5F7FA`
- 表头文字: 14px，字重 600
- 行高: 52px
- 斑马纹: 奇数行 `#FFFFFF`，偶数行 `#FAFBFC`
- 悬停背景: `#F5F7FA`

### 5.4 卡片 (Card)

- 背景: `#FFFFFF`
- 边框: 1px solid `#EBEEF5`
- 圆角: 8px
- 阴影: `0 2px 12px rgba(0, 0, 0, 0.08)`
- 内边距: 20px

### 5.5 状态标签 (Tag)

| 类型 | 背景色 | 文字色 |
|------|--------|--------|
| 成功 | `#E6F7EF` | `#00A854` |
| 警告 | `#FFF7E6` | `#FA8C15` |
| 错误 | `#FFF1F0` | `#F5222D` |
| 信息 | `#E6F7FF` | `#1890FF` |
| 默认 | `#F4F4F5` | `#606266` |

## 六、布局规范

### 6.1 管理后台布局

```
+--------------------------------------------------+
|  顶部导航栏 (64px height)                         |
+--------+-----------------------------------------+
| 侧边栏  |  内容区域                              |
| 240px  |  - 面包屑 (40px)                       |
|        |  - 页面标题                            |
|        |  - 筛选条件                            |
|        |  - 表格/表单                           |
|        |  - 分页                                |
+--------+-----------------------------------------+
```

### 6.2 侧边栏导航

- 宽度: 240px (展开) / 64px (收起)
- 背景: `#001529` (深色主题)
- 菜单项高度: 48px
- 菜单项选中背景: `#0066FF` (20% 透明度)
- 菜单项选中左边框: 3px `#0066FF`

### 6.3 栅格系统

- 页面栅格: 24 栏
- 常用布局: 12 + 12, 8 + 8 + 8, 6 + 6 + 6 + 6
- 卡片栅格: 4 栏 (每行显示4个)

## 七、页面设计

### 7.1 列表页通用结构

1. **页面标题区**
   - 标题 (28px, 字重 600)
   - 描述文字 (可选, 14px, 辅助色)

2. **筛选条件区**
   - 行高: 40px
   - 筛选项间距: 16px
   - 按钮右对齐

3. **操作栏**
   - 右侧: 新建按钮 (主按钮)
   - 左侧: 批量操作 (次按钮)

4. **数据表格**
   - 固定表头
   - 可选择
   - 操作列固定右侧

5. **分页**
   - 位置: 底部居右
   - 默认每页: 10 条

### 7.2 详情页通用结构

1. **页面标题区**
   - 标题 + 状态标签
   - 操作按钮 (右侧)

2. **基本信息卡片**
   - 2-3 列布局

3. **详细信息**
   - 卡片式组织
   - 时间线 (流程类)

4. **操作记录**
   - 底部展示
   - 时间倒序

### 7.3 表单页通用结构

1. **页面标题区**
   - 标题
   - 步骤条 (如需)

2. **表单区域**
   - 卡片式组织
   - 2 列布局 (桌面)
   - 1 列布局 (移动)

3. **底部操作栏**
   - 固定底部
   - 取消 + 提交按钮

## 八、保函特定设计

### 8.1 保函申请流程

```
[选择保函类型] → [填写申请信息] → [上传材料] → [确认提交]
     1             2               3            4
```

- 步骤条显示当前进度
- 每个步骤可保存草稿
- 材料上传支持拖拽

### 8.2 保函状态流转

```
                    ┌──────────┐
                    │  待提交   │
                    └─────┬────┘
                          │ 提交
                          ▼
                    ┌──────────┐
         ┌───────── │  待审核   │ ─────────┐
         │          └─────┬────┘          │
         │                │ 审核通过      │
         │                ▼               │
    审核驳回         ┌──────────┐          │
         │         │ 审核通过  │          │
         │         │ 待缴费   │          │
         │         └─────┬────┘          │
         │               │ 缴费完成       │
         │               ▼               │
         │         ┌──────────┐          │
         └─────────│  已出具   │◄─────────┘
                   └─────┬────┘
                         │ 生效
                         ▼
                   ┌──────────┐
                   │  已生效   │
                   └─────┬────┘
                         │ 到期/退保
                         ▼
                   ┌──────────┐
                   │  已终止   │
                   └──────────┘
```

### 8.3 保函文档展示

- PDF/OFD 预览组件
- 电子签章可视化
- 二维码验证标识
- 下载按钮固定位置

## 十、可访问性 (Accessibility)

### 10.1 颜色对比度

- 文字与背景对比度: 最小 4.5:1
- 大文字对比度: 最小 3:1
- 组件边框与背景: 最小 3:1

### 10.2 交互状态

- 所有交互元素有 `:focus` 视觉状态
- 焦点环: 2px `#0066FF`，偏移 2px
- 按钮点击有 `:active` 状态

### 10.3 屏幕阅读器

- 所有图片有 `alt` 文字
- 表格有表头关联
- 表单字段有正确关联的 `label`
