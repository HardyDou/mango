## 三、交互流程

### 3.1 登录注册流程

```
欢迎页 → 登录页 → 快捷登录页/注册页 → 首页
                        ↓
                   忘记密码页
```

### 3.2 投保申请流程

```
首页 → 选择保函类型 → 选择交易平台 → 选择出函机构
     → 项目信息填写 → 企业信息填写 → 材料上传
     → 费用预览 → 投保成功 → 投保进度列表
```

### 3.3 核心用户流程

```
登录 → 首页 → 投保申请 → 进度查询
     → 保函管理 → 消息通知 → 个人中心
```
