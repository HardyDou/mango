## 一、功能概述

本章描述电子保函系统批量操作功能的需求，提供对大量投保申请、保函的批量处理能力，提升机构用户工作效率。

---

## 二、批量操作场景

### 2.1 适用角色

| 角色 | 适用场景 |
|------|---------|
| 担保公司 | 批量投保、批量查询、批量导出 |
| 出函机构 | 批量审核、批量出函、批量退保处理 |
| 系统管理员 | 批量数据导入、批量状态更新 |

### 2.2 批量操作类型

| 操作类型 | 适用场景 | 优先级 |
|---------|---------|-------|
| 批量投保申请 | 一次性提交多个投保申请 | P1 |
| 批量审核 | 一次性审核多个投保申请 | P0 |
| 批量出函 | 一次性出具多个保函 | P0 |
| 批量查询 | 一次性查询多个保函状态 | P1 |
| 批量导出 | 一次性导出多个保函数据 | P0 |
| 批量退保处理 | 一次性处理多个退保申请 | P1 |
| 批量数据导入 | 批量导入企业信息、项目信息 | P1 |
| 批量通知 | 一次性发送多个通知 | P2 |

## 三、批量投保申请

### 3.1 功能描述

支持担保公司用户一次性提交多个投保申请，适用于同一企业多个项目同时投标的场景。

### 3.2 操作流程

```
选择项目 → 填写企业信息 → 上传材料 → 确认提交 → 批量投保
```

### 3.3 功能需求

| 功能 | 需求描述 |
|------|---------|
| 模板下载 | 下载批量投保Excel模板 |
| Excel导入 | 支持导入Excel文件批量提交 |
| 数据校验 | 导入前进行数据格式校验 |
| 预览确认 | 导入后预览确认所有申请 |
| 批量提交 | 确认后一次性提交所有申请 |
| 结果反馈 | 显示提交成功/失败明细 |

### 3.4 Excel模板字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 项目名称 | 文本 | 是 | 招标项目名称 |
| 项目编号 | 文本 | 是 | 招标项目编号 |
| 保函类型 | 下拉选择 | 是 | 投标/履约/预付款/质量 |
| 受益人 | 文本 | 是 | 招标单位名称 |
| 保函金额 | 数字 | 是 | 担保金额 |
| 保函有效期 | 日期 | 是 | 截止日期 |
| 出函机构 | 文本 | 是 | 出函机构名称 |
| 联系人 | 文本 | 是 | 联系人姓名 |
| 联系电话 | 文本 | 是 | 联系人手机号 |

---

## 四、批量审核

### 4.1 功能描述

支持出函机构用户一次性审核多个投保申请，提升审核效率。

### 4.2 操作流程

```
选择待审核申请 → 批量预览 → 选择审核结果 → 填写审核意见 → 提交审核
```

### 4.3 功能需求

| 功能 | 需求描述 |
|------|---------|
| 条件筛选 | 按状态、时间、机构等筛选待审核申请 |
| 批量选择 | 支持全选、按页选择、自定义选择 |
| 批量预览 | 批量查看选中申请的详细信息 |
| 统一审核 | 选择统一的审核结果（通过/拒绝） |
| 个别处理 | 支持对个别申请单独处理 |
| 审核意见 | 批量填写统一的审核意见 |
| 结果反馈 | 显示审核成功/失败明细 |

### 4.4 批量审核规则

| 规则 | 说明 |
|------|------|
| 数量限制 | 单次批量审核最多100条 |
| 状态限制 | 仅可批量审核状态为"待审核"的申请 |
| 权限限制 | 当前用户有权限审核的机构申请 |

---

## 五、批量出函

### 5.1 功能描述

支持出函机构用户一次性出具多个保函，提升出函效率。

### 5.2 操作流程

```
选择待出函申请 → 批量预览 → 确认出函信息 → 批量出函 → 结果反馈
```

### 5.3 功能需求

| 功能 | 需求描述 |
|------|---------|
| 条件筛选 | 按审核通过时间、机构等筛选待出函申请 |
| 批量选择 | 支持全选、按页选择、自定义选择 |
| 批量预览 | 批量查看选中申请的出函信息 |
| 出函确认 | 确认批量出函信息 |
| 进度显示 | 显示批量出函进度 |
| 结果反馈 | 显示成功/失败明细，支持下载失败清单 |

### 5.4 批量出函规则

| 规则 | 说明 |
|------|------|
| 数量限制 | 单次批量出函最多50条 |
| 状态限制 | 仅可批量出函状态为"审核通过"的申请 |
| 模板限制 | 使用统一的保函模板 |
| 生成顺序 | 按选中顺序生成，可调整顺序 |

## 六、批量查询

### 6.1 功能描述

支持用户一次性查询多个保函的状态，适用于大批量项目管理的场景。

### 6.2 功能需求

| 功能 | 需求描述 |
|------|---------|
| Excel导入查询 | 导入Excel批量查询保函状态 |
| 编号列表查询 | 输入多个保函编号批量查询 |
| 条件组合查询 | 按项目名称、时间范围等条件批量查询 |
| 结果导出 | 批量导出查询结果为Excel |

### 6.3 查询结果字段

| 字段 | 说明 |
|------|------|
| 保函编号 | 电子保函编号 |
| 项目名称 | 对应项目名称 |
| 状态 | 保函当前状态 |
| 出函时间 | 出函日期 |
| 到期时间 | 到期日期 |
| 出函机构 | 出函机构名称 |

## 七、批量导出

### 7.1 功能描述

支持用户一次性导出多个保函的相关数据，适用于数据统计、存档等场景。

### 7.2 功能需求

| 功能 | 需求描述 |
|------|---------|
| 条件筛选 | 按各种条件筛选需要导出的数据 |
| 字段选择 | 选择需要导出的字段 |
| 格式选择 | 支持Excel、CSV、PDF格式 |
| 批量下载 | 支持打包下载 |
| 导出记录 | 记录导出历史，支持重新下载 |

### 7.3 导出数据类型

| 数据类型 | 说明 |
|---------|------|
| 投保申请数据 | 投保申请相关信息 |
| 保函数据 | 保函基本信息 |
| 审核数据 | 审核流程记录 |
| 财务数据 | 保费、退款等财务数据 |
| 统计数据 | 汇总统计数据 |

### 7.4 导出规则

| 规则 | 说明 |
|------|------|
| 数量限制 | 单次导出最多10000条 |
| 文件大小 | 单个文件不超过50MB |
| 打包限制 | 超过50MB自动分包 |

## 九、批量数据导入

### 9.1 功能描述

支持系统管理员批量导入企业信息、项目信息等基础数据。

### 9.2 导入数据类型

| 数据类型 | 说明 |
|---------|------|
| 企业信息 | 批量导入企业基本信息 |
| 项目信息 | 批量导入项目信息 |
| 机构信息 | 批量导入出函机构信息 |
| 费率配置 | 批量导入费率配置 |

### 9.3 功能需求

| 功能 | 需求描述 |
|------|---------|
| 模板下载 | 下载各类型数据导入模板 |
| 数据校验 | 导入前进行数据格式和逻辑校验 |
| 重复检测 | 检测重复数据并提示 |
| 预览确认 | 预览导入数据，确认后执行 |
| 导入结果 | 显示导入成功/失败明细 |
| 错误日志 | 记录导入失败原因，支持查看 |

### 9.4 导入规则

| 规则 | 说明 |
|------|------|
| 文件格式 | 仅支持Excel格式 |
| 数量限制 | 单次导入最多5000条 |
| 必填校验 | 必填字段不能为空 |
| 格式校验 | 数据格式必须符合要求 |

## 十、批量操作通用设计

### 10.1 界面设计

| 设计要素 | 要求 |
|---------|------|
| 操作入口 | 在列表页提供批量操作入口 |
| 选择方式 | 支持Checkbox多选 |
| 进度显示 | 长时间操作显示进度条 |
| 结果反馈 | 完成后显示成功/失败明细 |
| 撤销机制 | 支持撤销进行中的批量操作 |

### 10.2 错误处理

| 错误类型 | 处理方式 |
|---------|---------|
| 数据校验失败 | 列出具体失败行和原因 |
| 部分失败 | 成功后继续执行，失败部分记录 |
| 全部失败 | 回滚操作，提示错误原因 |
| 超时 | 支持断点续传 |

### 10.3 安全控制

| 控制点 | 说明 |
|-------|------|
| 权限控制 | 仅授权用户可进行批量操作 |
| 数量限制 | 单次操作有数量上限 |
| 操作日志 | 记录所有批量操作日志 |
| 审计追溯 | 支持审计批量操作记录 |
