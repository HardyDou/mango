## 五、技术实现

### 5.1 技术栈

| 层级 | 技术选型 |
|------|---------|
| 后端框架 | Spring Cloud |
| 数据库 | MySQL 8.0 + Redis |
| 缓存 | Redis Cluster |
| 文件存储 | 本地存储（后期对象存储） |
| PDF生成 | iText/OpenPDF |
| 图表 | ECharts |
| 报表导出 | Apache POI |

### 5.2 核心模块设计

#### 退保模块

```
数据库表：
- surrender_apply（退保申请）
- surrender_audit（退退审记录）
- surrender_refund（退款记录）

核心服务：
- SurrenderApplyService（退保申请）
- SurrenderAuditService（审核流程）
- RefundService（退款处理）
- FeeCalculateService（费用计算）
```

#### 保函管理模块

```
数据库表：
- guarantee_extension（延期记录）
- guarantee_change（变更记录）
- guarantee_termination（终止记录）

核心服务：
- GuaranteeQueryService（保函查询）
- GuaranteeVerifyService（验真服务）
- GuaranteeExtensionService（延期服务）
- GuaranteeChangeService（变更服务）
- GuaranteeTerminationService（终止服务）
```

#### 发票模块

```
数据库表：
- invoice_apply（开票申请）
- invoice_issue（开票记录）
- invoice_redflush（红冲记录）

核心服务：
- InvoiceApplyService（开票申请）
- InvoiceGenerateService（发票生成）
- InvoiceRedflushService（红冲服务）
- InvoiceDeliveryService（交付服务）
```

#### 报表模块

```
数据库表：
- report_template（报表模板）
- report_data（报表数据）
- report_schedule（定时报表）

核心服务：
- ReportGenerateService（报表生成）
- StatisticsService（数据统计）
- ChartService（图表服务）
- ExportService（导出服务）
```
