### 2.2 表格列定义

| 列名 | 宽度 | 内容 |
|------|------|------|
| 复选框 | 48px | 批量选择（可选） |
| 保函编号 | 160px | BG202601010001 |
| 项目名称 | 200px | 模糊显示，超出省略 |
| 保函类型 | 100px | 投标/履约/预付款/质量 |
| 投保人 | 150px | 企业名称 |
| 受益人 | 150px | 企业名称 |
| 担保金额 | 120px | 金额，右对齐，千位分隔 |
| 保费 | 100px | 金额，右对齐 |
| 状态 | 100px | 状态标签 |
| 出函日期 | 120px | YYYY-MM-DD |
| 到期日期 | 120px | YYYY-MM-DD |
| 操作 | 120px | 查看/下载 |

### 2.3 状态标签规范

| 状态 | 背景色 | 文字色 |
|------|--------|--------|
| 待审核 | #FFF7E6 | #FA8C15 |

| 已出具 | #E6F7EF | #00A854 |
| 生效中 | #E6F0FF | #0066FF |
| 已过期 | #F4F4F5 | #909399 |

### 2.4 高级筛选

| 字段 | 类型 | 规范 |
|------|------|------|
| 保函编号 | 输入框 | 精确匹配 |
| 项目名称 | 输入框 | 模糊匹配 |
| 保函类型 | 下拉多选 | 投标/履约/预付款/质量 |
| 担保公司 | 下拉选择 | 关联数据 |
| 出函机构 | 下拉选择 | 关联数据 |
| 保函状态 | 标签选择 | 多选 |
| 出函日期 | 日期范围 | 开始-结束 |
| 到期日期 | 日期范围 | 开始-结束 |

### 4.2 验真结果状态

| 结果 | 样式 |
|------|------|
| 验真成功 | 绿色勾选图标 + 绿色边框卡片 |
| 验真失败 | 红色叉号图标 + 红色边框卡片 + "该保函不存在或已失效" |

### 4.3 验真方式卡片

| 方式 | 图标 | 说明 |
|------|------|------|
| 编号验真 | 搜索图标 | 输入保函编号查询 |
| 二维码验真 | 二维码图标 | 扫描保函上的二维码 |
| 哈希验真 | 锁图标 | 验证OFD文件哈希值 |
