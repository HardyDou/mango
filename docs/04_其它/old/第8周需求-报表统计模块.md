# 第8周需求 - 报表统计模块

**版本**: V1.0
**制定日期**: 2026-03-14
**制定人**: 电子保函业务专家
**状态**: 已优化

---

## 1. 业务概述

报表统计功能提供电子保函业务的全面数据统计分析，支持多维度数据分析和导出。

### 1.1 报表类型

| 类型 | 说明 |
|------|------|
| 保函统计 | 保函数量、金额、类型分布 |
| 投保统计 | 投保申请、审核、通过率 |
| 财务报表 | 保费收入、退费、利润 |
| 机构统计 | 各机构业务量排名 |

### 1.2 统计维度

| 维度 | 说明 |
|------|------|
| 时间维度 | 按日/周/月/季/年统计 |
| 机构维度 | 按担保公司/出函机构统计 |
| 类型维度 | 按保函类型统计 |
| 状态维度 | 按保函状态统计 |
| 区域维度 | 按地区统计 |

## 2. 功能需求

### 2.1 保函统计

**功能描述**: 统计保函业务数据

**统计指标**:

| 指标 | 说明 |
|------|------|
| 保函总量 | 累计出具保函数量 |
| 保函总金额 | 累计担保金额 |
| 在保金额 | 当前生效保函总额 |
| 今日新增 | 今日新增保函数量 |
| 本周新增 | 本周新增保函数量 |
| 本月新增 | 本月新增保函数量 |
| 在保数量 | 当前生效保函数量 |

**图表展示**:
- 保函数量趋势图（折线图）
- 保函金额趋势图（折线图）
- 保函类型分布（饼图）
- 保函状态分布（饼图）
- 机构保函排行（柱状图）

**筛选条件**:
- 时间范围：今日/本周/本月/本季度/本年/自定义
- 机构：全部/指定机构
- 类型：全部/投标/履约/预付款/质量
- 状态：全部/生效/已退保/已过期

### 2.2 投保统计

**功能描述**: 统计投保业务数据

**统计指标**:

| 指标 | 说明 |
|------|------|
| 投保申请量 | 累计投保申请数量 |
| 审核通过量 | 审核通过数量 |
| 审核通过率 | 通过量/申请量 |
| 投保退海量 | 退保申请数量 |
| 退保率 | 退保量/投保量 |
| 平均出函时间 | 申请到出函平均天数 |

**图表展示**:
- 投保申请趋势图（折线图）
- 审核通过率趋势图（折线图）
- 各类型投保占比（饼图）
- 退保原因分布（条形图）
- 机构投保排行（柱状图）

### 2.3 财务报表

**功能描述**: 统计财务收入数据

**统计指标**:

| 指标 | 说明 |
|------|------|
| 保费总收入 | 累计收取保费 |
| 本期保费 | 本期收取保费 |
| 退费总额 | 累计退费金额 |
| 净保费收入 | 保费收入-退费 |
| 发票开具量 | 累计开具发票 |
| 发票金额 | 累计开票金额 |

**图表展示**:
- 保费收入趋势图（折线图）
- 收入支出对比图（柱状图）
- 各类型收入占比（饼图）
- 月度收支趋势（面积图）

**收入明细**:
- 保费收入（按保函类型）
- 延期费收入
- 其他收入

**支出明细**:
- 退费支出
- 赔付支出
- 其他支出

### 2.4 机构统计

**功能描述**: 统计各机构业务数据

**统计维度**:

| 维度 | 说明 |
|------|------|
| 担保公司统计 | 各担保公司业务量、金额 |
| 出函机构统计 | 各银行/保险公司业务量 |
| 机构对比 | 机构间业务对比分析 |

**展示字段**:

| 字段 | 说明 |
|------|------|
| 机构名称 | 担保公司/出函机构名称 |
| 保函数量 | 出具保函数量 |
| 担保金额 | 担保总金额 |
| 保费收入 | 收取保费 |
| 市场占比 | 占总业务比例 |
| 排名 | 业务量排名 |

### 2.5 数据导出

**功能描述**: 导出统计数据

**导出格式**:
- Excel (.xlsx)
- PDF
- CSV

**导出内容**:
- 原始数据
- 汇总数据
- 图表

## 3. 接口需求

### 3.1 保函统计接口

**接口路径**: `GET /api/v1/statistics/guarantee`

**请求参数**:
```json
{
  "startDate": "string",         // 开始日期
  "endDate": "string",           // 结束日期
  "insurerId": "string",        // 担保公司ID
  "issuerId": "string",         // 出函机构ID
  "guaranteeType": "string",    // 保函类型
  "granularity": "string"       // 粒度：DAY/WEEK/MONTH/QUARTER/YEAR
}
```

**响应参数**:
```json
{
  "code": "200",
  "message": "success",
  "data": {
    "total": {
      "count": "number",        // 保函总量
      "amount": "number",       // 保函总金额
      "inForceAmount": "number", // 在保金额
      "inForceCount": "number"  // 在保数量
    },
    "trend": [
      {
        "date": "string",
        "count": "number",
        "amount": "number"
      }
    ],
    "typeDistribution": [
      {
        "type": "string",
        "count": "number",
        "amount": "number"
      }
    ],
    "statusDistribution": [
      {
        "status": "string",
        "count": "number"
      }
    ]
  }
}
```

### 3.2 投保统计接口

**接口路径**: `GET /api/v1/statistics/application`

### 3.3 财务报表接口

**接口路径**: `GET /api/v1/statistics/finance`

### 3.4 机构统计接口

**接口路径**: `GET /api/v1/statistics/org`

### 3.5 数据导出接口

**接口路径**: `POST /api/v1/statistics/export`

**请求参数**:
```json
{
  "type": "string",             // GUARANTEE/APPLICATION/FINANCE/ORG
  "startDate": "string",
  "endDate": "string",
  "format": "string",           // EXCEL/PDF/CSV
  "columns": ["string"]         // 导出字段
}
```

## 8. 业务规则补充

### 8.1 统计数据更新频率

| 报表类型 | 更新频率 | 说明 |
|----------|---------|------|
| 实时看板 | 实时 | 当前数据 |
| 日统计 | 每日0点 | T+1 |
| 周统计 | 每周一 | 上周数据 |
| 月统计 | 每月1日 | 上月数据 |
| 年统计 | 每年1月 | 上年数据 |

**定时任务配置**：
- 统计数据汇总：每日 00:05 执行
- 报表数据生成：每日 00:30 执行
- 历史归档：每月1日执行

### 8.2 历史数据保留规则

| 数据类型 | 保留期限 | 存储方式 |
|----------|---------|----------|
| 原始交易数据 | 永久 | 热存储 |
| 统计数据 | 10年 | 热存储 |
| 归档数据 | 10年 | 冷存储 |
| 日志数据 | 1年 | 日志存储 |

### 8.3 自定义报表（V2.0规划）

**V1.0版本**：
- 不支持自定义报表
- 仅提供标准报表模板

**V2.0规划**：
- 支持自定义字段组合
- 支持自定义筛选条件
- 支持自定义图表类型
- 支持报表模板保存

### 8.4 财务系统对接（预留）

**接口预留**：
- 财务数据导出接口
- 标准财务凭证格式
- 对接用友、金蝶等主流财务系统
