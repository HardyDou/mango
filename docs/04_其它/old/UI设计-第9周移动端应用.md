## 一、移动端设计规范

### 1.1 移动端设计原则

| 原则 | 描述 |
|------|------|
| **简洁高效** | 移动端操作简便，减少输入，信息精炼 |
| **触摸友好** | 触摸区域≥44px，间距合理，支持手势 |
| **单手操作** | 关键操作在拇指易触达区域 |
| **快速响应** | 加载时间<3秒，操作反馈及时 |

### 1.2 移动端色彩体系

沿用Web端色彩体系，增加移动端特定配色：

| 色彩名称 | 色值 | 用途 |
|----------|------|------|
| 主色 | `#0066FF` | 主要按钮、强调元素 |
| 主色渐变 | `linear-gradient(135deg, #0066FF, #0052CC)` | 按钮背景 |
| 背景色 | `#F5F7FA` | 页面背景 |
| 卡片白 | `#FFFFFF` | 卡片、弹窗 |
| 分割线 | `#EBEEF5` | 列表分割 |
| 文字主色 | `#1F1F1F` | 标题 |
| 文字次色 | `#606266` | 正文 |
| 文字辅助 | `#909399` | 提示文字 |

### 1.3 移动端字体

| 字号 | 行高 | 用途 |
|------|------|------|
| 20px | 28px | 页面大标题 |
| 18px | 26px | 卡片标题 |
| 16px | 24px | 正文 |
| 14px | 20px | 辅助文字 |
| 12px | 16px | 标签、小字 |

### 1.4 移动端间距

| 间距 | 数值 | 用途 |
|------|------|------|
| xs | 4px | 标签内间距 |
| sm | 8px | 图标与文字间距 |
| md | 16px | 列表项间距 |
| lg | 24px | 区块间距 |
| xl | 32px | 页面边距 |

### 1.5 移动端组件

#### 1.5.1 导航栏 (Navigation Bar)

- 高度: 44px (普通) / 88px (包含安全区域)
- 背景: `#FFFFFF`
- 底部阴影: `0 1px 0 rgba(0,0,0,0.05)`
- 标题居中，字体16px，字重600

#### 1.5.2 标签栏 (Tab Bar)

- 高度: 49px (普通) / 83px (包含安全区域)
- 背景: `#FFFFFF`
- 固定底部
- 图标尺寸: 24px
- 选中状态: 主色 `#0066FF`
- 未选中状态: `#909399`

#### 1.5.3 列表项

- 高度: 最小56px
- 左侧图标: 24px
- 右侧箭头: 12px
- 点击反馈: 背景色 `rgba(0,0,0,0.03)`

#### 1.5.4 按钮

| 类型 | 样式 | 尺寸 |
|------|------|------|
| 主按钮 | 主色填充，圆角22px | 高度44px，全宽 |
| 次按钮 | 主色边框，圆角22px | 高度44px，全宽 |
| 文字按钮 | 无背景，字重500 | 高度44px |
| 危险按钮 | 红色填充 `#F5222D` | 高度44px，全宽 |

#### 1.5.5 输入框

- 高度: 44px
- 边框: 1px solid `#DCDFE6`
- 圆角: 8px
- 聚焦边框: 1px solid `#0066FF`
- 背景: `#F5F7FA`

#### 1.5.6 状态标签

| 状态 | 背景色 | 文字色 |
|------|--------|--------|
| 待审核 | `#FFF7E6` | `#FA8C15` |
| 审核中 | `#E6F7FF` | `#1890FF` |
| 已出函 | `#E6F7EF` | `#00A854` |
| 已生效 | `#E6F0FF` | `#0066FF` |
| 已过期 | `#F4F4F5` | `#909399` |
| 已退保 | `#FFF1F0` | `#F5222D` |
| 已驳回 | `#FFF1F0` | `#F5222D` |

## 三、交互规范

### 3.1 手势操作

| 手势 | 适用场景 |
|------|---------|
| 点击 | 选择、确认、跳转 |
| 长按 | 显示操作菜单 |
| 左滑 | 快速查看、删除 |
| 下拉 | 刷新列表 |
| 上拉 | 加载更多 |

### 3.2 加载状态

- 列表加载: 显示骨架屏
- 按钮加载: 显示loading图标，禁用点击
- 页面加载: 显示全屏loading或骨架屏

### 3.3 错误处理

- 输入错误: 实时校验，红字提示
- 网络错误: toast提示，"点击重试"
- 服务错误: 弹窗提示，"稍后重试"

### 3.4 空状态

- 无数据: 显示插画+提示文字+操作按钮
- 示例: "暂无投保记录，快去申请吧"
