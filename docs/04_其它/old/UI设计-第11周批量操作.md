## 一、批量操作设计概述

### 1.1 设计范围

本章设计涵盖以下批量操作场景的UI：
- 批量投保申请
- 批量审核
- 批量出函
- 批量查询
- 批量导出
- 批量退保处理
- 批量数据导入

## 九、交互规范

### 9.1 数量限制提示

| 操作类型 | 数量限制 | 超出提示 |
|---------|---------|---------|
| 批量审核 | 100条/次 | "单次最多审核100条" |
| 批量出函 | 50条/次 | "单次最多出函50条" |
| 批量导出 | 10000条/次 | "单次最多导出10000条" |
| 批量导入 | 5000条/次 | "单次最多导入5000条" |

### 9.2 进度显示

- 进度条显示当前处理进度
- 显示"已处理/总数"
- 显示当前处理项
- 支持取消操作

### 9.3 错误处理

- 部分失败: 成功后继续，失败记录详细原因
- 全部失败: 弹窗提示，显示所有错误
- 取消操作: 确认弹窗，已处理部分保留
