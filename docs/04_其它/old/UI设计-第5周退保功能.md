## 二、退保申请页面

### 2.1 页面结构

```
+--------------------------------------------------+
|  顶部导航栏                                       |
+--------+-----------------------------------------+
| 侧边栏  |  内容区域                              |
|        |  +----------------------------------+  |
|        |  | 页面标题: 申请退保                |  |
|        |  +----------------------------------+  |
|        |  | 步骤条: [1]选择保函>[2]退保原因  |  |
|        |  |          >[3]上传材料>[4]确认    |  |
|        |  +----------------------------------+  |
|        |  |                                  |  |
|        |  |        表单区域                  |  |
|        |  |                                  |  |
|        |  +----------------------------------+  |
|        |  | 底部操作栏: [取消] [提交申请]    |  |
|        |  +----------------------------------+  |
+--------+-----------------------------------------+
```

### 2.2 步骤一：选择保函

**布局**: 卡片式列表

| 元素 | 规范 |
|------|------|
| 保函卡片 | 白色背景，圆角8px，阴影 |
| 卡片内容 | 保函编号、保函类型、项目名称、投保人、保费金额、状态标签 |
| 状态标签 | 已出具(#00A854)/生效中(#0066FF) |
| 选中态 | 边框色 #0066FF，2px |
| 搜索框 | 顶部固定，支持保函编号、项目名称搜索 |

### 2.3 步骤二：退保原因

**布局**: 2列表单

| 字段 | 类型 | 规范 |
|------|------|------|
| 退保原因 | 下拉选择 | 合同履行完毕/部分履行/司法判决/提前解除 |
| 退保说明 | 文本域 | 可选，200字以内 |
| 退款方式 | 单选 | 银行转账/保证金释放 |

### 2.4 步骤三：上传材料

**布局**: 拖拽上传区

| 元素 | 规范 |
|------|------|
| 上传区域 | 虚线边框，拖拽支持 |
| 文件类型 | 图片/PDF |
| 最大数量 | 10个文件 |
| 单个大小 | 最大10MB |
| 预览 | 图片缩略图，PDF图标+文件名 |
| 删除 | 右上角 X 按钮 |

### 2.5 步骤四：确认提交

**布局**: 确认信息卡片

| 区域 | 内容 |
|------|------|
| 退保信息摘要 | 保函编号、退保原因、退费金额（系统计算） |
| 退款账户 | 开户行、账户名、账号（脱敏显示） |
| 手续费 | 金额（根据退保类型计算） |
| 预计退款 | 退费金额 - 手续费 |
