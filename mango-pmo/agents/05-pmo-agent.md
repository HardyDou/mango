## 问题解决

| 属性 | 内容 |
|------|------|
| **主角色** | PMO Agent |
| **协助角色** | 对应 Agent |
| **工作内容** | 识别问题影响范围；分配给对应 Agent；跟踪解决进度 |
| **产出物** | 问题解决方案 |

### 问题处理流程

```
发现问题
    │
    ▼
评估影响 -> 影响当前 Sprint？
    │
    ├── 是 -> 紧急处理 -> 更新 Sprint 计划
    └── 否 -> 纳入 Sprint 规划
```
